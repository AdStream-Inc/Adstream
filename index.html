Using Grids

The grid system is fully fluid and adaptable to its enviroment. The system also includes support for nested grids and grid offsets. The only required markup to include a grid is to wrap it in a div with the class of .row as seen in the example to the right.

See Grid System In Action 
<div class="container">
	<h1> Hello World </h1>
	<div class="row">
		<div class="grid6">
			…
		</div>
		<div class="grid4 offset2">
			…
		</div>
	</div>
</div>

Headings

Headings use a unique class system that allows them to be sized according to its semantic value, meaning can make a h3 to look like a h1 with just a simple class. All sizes are created to allow for a generous amount of visual hierarchy, there are also sizing classes when an extremeley large font size.

<h1 class="h3">I look like a h3</h3>
<h1 class="huge">Im a huge header!</h1>

CSS Abstractions

Media Object

An extremely versatile object that can be used for all sorts of applications. See the Media Object for more info.

<div class="media">
	<a class="img">< img src="…" /></a>
	<div class="bd">
		…
	</div>
</div>

Nav Object

The nav abstraction can be used anytime you want to format an list of items. It supports both horizontal and vertical lists. It also has support for nested dropdown menus.

<ul class="nav">
	<li>< a href="…"></a></li>
	<li class="dropdown">
		<a href="…">…</a>
			<ul class="dropdown-content">
				…
			</ul>
	</li>
</ul>

Island Object

The island abstraction is used when you want to section off an area of content for things like promos or toolbars. Can be extended with classes such as .brand or .twitter.

<div class="island">
	…
</div>

Button Obect

The button object can be applied to just about any element you want, making it an extremely robust component to work with. Since there is no base styles added to it, besides padding for spacing, use it in conjunction with things such as .button-blue or .brand.

<a class="button brand">Anchor</a>
<input type="submit" value="Input" class="button brand" />
<button class="button brand">Button</button>
<span class="button brand">Span</span>

Utility Classes

Removing Vertical Margins

Use this class if you wish to remove the vertical rhythm margin from various components such as the .nav abstraction.

.no-vertical-margin{ margin-bottom: 0; }

<ul class="nav no-vertical-margin"></ul>

Floating Components

These are pretty self explanatory, use these classes to quickly float an item or clear its floats.

.left{ float:left; }
.right{ float:right; }
.clear{ clear:both; float:none; }

Aligning Text

Pretty straightfoward as well, use these classes to align text items.

.text-left{ text-align:left; }
.text-right{ text-align:right; }
.text-center{ text-align:center; }

SEO Friendly Accessibility

If you wish to hide a element but keep it SEO friendly use this class to simply remove it from the view of users but remain for crawlers.

.accessibility{ position:absolute; left:-99999px; }

<img src="..." alt="A hidden image" class="accessibility" />
